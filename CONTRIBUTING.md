# Contributing to Coffee Grind

Thank you for contributing to our project! Please follow the guidelines below.

## Step 1: Fork the repository

Fork the repository to your own GitHub account.

## Step 2: Clone the repository

Run the following command to clone your fork:

```
git clone https://github.com/montenays82-jpg/coffee-grind.git
```

## Step 3: Navigate into your cloned repository

Change into the project directory:

```
cd coffee-grind
```

## Step 4: Create a new branch

It's a good practice to create a new branch for your changes.

```bash
git checkout -b <your-branch-name>
```

## Step 5: Make your changes

Make the necessary changes to the code, then commit them:

```bash
git commit -m "Your commit message"
```

## Step 6: Push your changes

Push your changes to your forked repository:

```bash
git push origin <your-branch-name>
```

## Step 7: Create a Pull Request

Now, navigate to your original repository and create a Pull Request. Thank you for contributing!
